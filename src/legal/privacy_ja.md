# kura プライバシーポリシー

最終更新日: 2026年4月20日

## 1. 基本方針

kura（以下「本ソフトウェア」）はゼロ知識設計のパスワードマネージャーです。本ソフトウェアは開発者が運用するサーバーを一切持たないため、開発者がユーザーの個人情報を収集・保存・送信することはありません。

## 2. 開発者が収集する情報

本ソフトウェアの開発者は、ユーザーから一切の情報を収集しません。これには以下を含みますが、これらに限定されません。

- マスターパスワード
- vault に保存されたパスワードアイテム（ログイン情報、ノート、カスタムフィールド等）
- S3 ストレージの認証情報（Access Key / Secret Key 等）
- ユーザーが訪問したウェブサイトの URL、ドメイン、入力内容
- デバイス情報、利用ログ、クラッシュレポート
- IP アドレス、位置情報
- メールアドレス、氏名等のアカウント情報

本ソフトウェアの利用にあたり、開発者への登録やアカウント作成は一切必要ありません。

## 3. ユーザーが入力する情報の扱い

以下の情報はユーザー自身が入力し、すべて暗号化されてユーザーが指定した S3 互換ストレージに保存されます。

- **マスターパスワード**: 暗号化鍵の導出にのみ使用され、ストレージには一切保存されません。端末のメモリ上で使用され、ロック時にゼロ埋めして破棄されます。
- **vault に保存するパスワードアイテム**: AES-256-GCM で暗号化された状態でのみストレージに保存されます。
- **S3 ストレージ設定（Access Key / Secret Key / エンドポイント等）**: マスターパスワード由来の鍵で暗号化され、お使いのデバイス内（ブラウザ拡張の場合は `browser.storage.local`、デスクトップの場合は Tauri store、Android の場合は Android Keystore）に保存されます。
- **リカバリーキー**: ユーザー自身で紙等に保管する設計です。デバイスにもストレージにも平文では保存されません。

## 4. 暗号化アーキテクチャ

- **暗号化方式**: AES-256-GCM による認証付き暗号化
- **鍵導出**: Argon2id（メモリハード関数）
- **鍵階層**: マスターパスワードから導出した KEK（Key Encryption Key）で DEK（Data Encryption Key）を暗号化。DEK で vault データを暗号化
- **鍵の所在**: 暗号化鍵はユーザーのデバイス上でのみ存在し、ストレージには暗号化された状態でのみ保存されます

## 5. 通信先と通信内容

本ソフトウェアが行う通信は以下に限定されます。

- ユーザーが明示的に設定した S3 互換エンドポイントへの vault 読み書き
- 拡張機能の場合、ブラウザが提供する API を介した同一デバイス内の通信

以下は一切行いません。

- 開発者が運用するサーバーへの通信
- Google Analytics 等の解析サービスへの送信
- 広告配信、クラッシュレポート収集、テレメトリ送信
- 第三者クラウドサービスへのデータ送信
- ユーザーが閲覧するウェブサイトの内容の外部送信

## 6. ブラウザ拡張機能の権限

Firefox および Chrome 向けの拡張機能では、以下の権限を使用します。

| 権限 | 用途 |
|------|------|
| `storage` | vault 設定・キャッシュをブラウザのローカルストレージに保存 |
| `alarms` | 自動ロック機能のためのタイマー |
| `clipboardWrite` / `clipboardRead` | パスワードのクリップボードコピーと一定時間後の自動消去 |
| `tabs` | 現在のタブの URL に対応するエントリを判定（URL のみ参照。閲覧履歴は取得しません） |
| `offscreen`（Chrome のみ） | Manifest V3 環境での WebAssembly 実行 |
| `<all_urls>` content script | ログインフォーム検出と自動入力。検出したフォームの内容を外部に送信することは一切ありません |

## 7. Cookie およびトラッカー

本ソフトウェアは Cookie を発行しません。解析・広告・トラッキング目的のスクリプトや SDK を一切含みません。

## 8. 第三者への情報提供

開発者がユーザーの情報にアクセスすることがないため、第三者に提供される情報はありません。ユーザーが選択した S3 互換ストレージサービスには、暗号化された vault ファイルのみがアップロードされます。ストレージサービスの利用規約は、各提供者の定めるところに従います。

## 9. 未成年者のプライバシー

本ソフトウェアは13歳未満のユーザーを対象としていません。開発者が個人情報を収集しないため、個別の同意取得手続きも行いません。

## 10. データの削除

- **vault データ**: ユーザーが S3 ストレージから vault ファイルを削除することで、暗号化されたデータも完全に消去されます。
- **ローカルキャッシュ**: 本ソフトウェアのアンインストール、またはブラウザの拡張機能設定から削除できます。
